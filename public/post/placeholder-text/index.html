Jan 4, 2026· JSON Alexander

It’s actually pretty sweet and easy to use, alliteration is just funny.

First of all, I’ve never used toml and struggled for 45 minutes to figure out that my pages weren’t displaying because I was using colons rather than equals to set menu parameters. toml stands for tom’s obvious markup language. I am dumb. I also struggled to understand the code flow of hugo, but after starting and playing with a few new themes I’ve learned that the documentation and robust template options make it ridiculously easy to use. The learning curve after day one was actually not that bad at all, though I doubt I’ll be writing many templates for the time being.
